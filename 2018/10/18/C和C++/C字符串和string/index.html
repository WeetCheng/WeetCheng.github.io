C字符串和string类

发表于 2018-10-18 | 分类于 编程 | | 阅读次数：

C风格字符串是指形如”hello world”这样的，以’\0’作为字符串结束符的传统的字符串。

string字符串则是指C++中的string类对象。string类对象本质上是对C风格字符串的封装，从而方便程序员使用。

常用C风格字符串函数

常用的C风格字符串处理函数有

函数名 作用
char *strcpy(char *dest, const char *src); 将字符串从src处复制到dest处
char *strncpy(char *dest, const char *src, size_t n); 同上，不过最多复制n个字节
void *memset(void *s, int c, size_t n); 将s处开始的n个字节设为c
void *memcpy(void *dest, const void *src, size_t n); 从src处复制n个字节到dest处
void *memmove(void *dest, const void *src, size_t n); 从src处复制n个字节到dest处
char *strcat(char *dest, const char *src); 将src的字符串接到dest后面
char *strncat(char *dest, const char *src, size_t n); 同上，不过最多接n个字节
int strcmp(const char *s1, const char *s2); 比较两个字符串，按ASCII比较
int strncmp(const char *s1, const char *s2, size_t n); 同上，不过最多比较n个字符
char *strstr(const char *haystack, const char *needle); 找出子串needle在主串haystack的位置
char *strcasestr(const char *haystack, const char *needle); 同上，不过忽略子串和主串大小写的不同

C风格字符串转换成数字

函数名 作用
int atoi(const char *nptr); 字符串转int
long atol(const char *nptr); 字符串转long
long long atoll(const char *nptr); 字符串转long long
double atof(const char *nptr); 字符串转double
double strtod(const char *nptr, char **endptr); 字符串转double
float strtof(const char *nptr, char **endptr); 字符串转float
long double strtold(const char *nptr, char **endptr); 字符串转long double
int sscanf(const char str, const char format, …); 字符串转各种类型数字
