DSP matlab常用函数

发表于 2018-11-08 | 更新于 2018-11-12 | 分类于 信号处理 | | 阅读次数：

生成信号

rand——用于生成均匀分布的信号

randn——生成高斯分布的信号

sinc——生成sinc信号

diric——生成周期的sinc信号

chirp——生成chirp信号(该信号的频率是时变的，与时间成正比)

信号的运算

conv——两个信号的卷积

xcorr——两个信号的相关函数（如果两个信号相同，则变成自相关函数）

fliplr——信号左右翻转

离散系统

y = filter(b, a, x)——由B(z)和A(z)决定离散系统H(z),输入信号x，得到输出信号y

h = impz(b, a, N)——根据B(z)和A(z)求出离散系统H(z)的单位抽样响应h(n),长度为N

[H, w] = freqz(b, a, N, 'whole', Fs)——求离散系统H(z)的频率响应$H(e^{j \omega})$,N是点数，Fs是抽样频率

zplane(b, a)——绘制离散系统H(z)的零极点图

[r, p, k] = residuez(b, a)——将离散系统H(z)的有理分式分解成多个简单有理分式的和

绘图相关

stem——绘制序列图

subplot(abc)——绘制figure分成a行b列，绘制第c幅

title，xlabel，ylable——图名，横坐标名，纵坐标名

axis——控制横纵坐标显示范围

set(gcf, ‘color’, ‘w’)——控制图形背景为白色
