采样定理

发表于 2018-11-24 | 更新于 2018-11-27 | 分类于 信号处理 |

时域脉冲序列及其频谱：
采样的意思就是时域上乘以下面的时域脉冲序列，根据卷积定理，时域乘以脉冲序列就等于频域卷积上频谱的脉冲序列。
又因为频谱与脉冲的卷积实际上是等于对频谱进行复制，平移和调幅，因此采样的结果就是将信号的频谱在频域的脉冲序列处复制一块出来。

关于采样定理的内容，可以参考知乎大佬 J Pan的专栏文章 如何理解离散傅里叶变换及Z变换

频谱图与时频图

频谱图：认为信号的频谱是固定的，不随时间改变。用fft计算

时频图：认为信号的频谱是随时间变化的。用sfft计算

chirp信号：频率随时间改变的正弦信号。

下面是在1s内频率从220均匀变化到440Hz的chirp信号的频谱图和时频图。

从时频图可以看出chirp信号的频率随时间线性增大。图中的马赛克反应了两个概念时间分辨率和频率分辨率，后面讲解。

时间分辨率与频率分辨率

上面的代码生成1s内从220Hz变化到440Hz的chirp信号，采样频率为11025，计算时频图时用512个点来计算sfft。

马赛克的宽表示时间分辨率，高表示频率分辨率。

时间分辨率是指用于计算sfft的每小段信号的长度（单位：秒）

频率分辨率是指sfft算出来的相邻两个频率的间隔，亦为基频（单位：Hz）

要想知道相邻两个频率的间隔是怎么算的，就必须了解DFT的原理

可以算出，

$$时间分辨率 = 512* \frac{1}{11025} = 0.046s$$
$$频率分辨率 = \frac{11025}{512} = 21.6Hz$$

假设DFT的计算点数为N，采样频率为fs，则

$$时间分辨率=N \cdot f_s$$
$$频率分辨率=f_s/N$$
从上面的公式可以看出，时间分辨率和频率分辨率互为倒数，因此两者相互制约，不可能同时提高，提高时间分辨率必然会降低频率分辨率，提高频率分辨率必然降低时间分辨率。
这种现象称为Gabor limit。

频谱混叠

发表于 2018-11-24 | 更新于 2018-11-27 | 分类于 信号处理 |

频谱混叠

频谱混叠是指信号的高频成分被搬移到低频成分。造成频谱混叠的主要原因跟采样频率有关，假设用10000Hz的采样频率去采样5500Hz和4500Hz的正弦波，得到的采样点是一样的，因此无法判断实际信号到底是4500Hz还是5500Hz，而通常都是取较小的那个，因此如果实际信号是5500Hz，用10000Hz的频率采样的话，会被误认为是4500Hz。

可检测的实际频率范围：根据频谱混叠的原理，可以知道如果实际信号的频率小于等于采样频率的一半，那么频率就没有搞错，如果频率大于采样频率的一半，就会搞错，因此可检测的实际频率范围是采样频率的一半。称为folding frequency

可以注意到 4500Hz+5500Hz=10000Hz(采样频率)。

频谱泄露

DFT背后有一个隐含的假设：信号是周期的，不过只截取了其中一小段来计算。如下图所示，假设我们取了正弦信号的一部分用于计算dft，由于dft的隐含假设，我们需要将这一段信号进行周期延拓。然后你可能会想当然地觉得就是正弦波的无限扩展，如第二幅图一样。很遗憾，dft的隐含条件是将拿来计算的那段信号首尾拼接，所以dft算的实际是第三幅图那样的信号。

因为在首尾拼接的时候，起点和终点的值不相等，所以拼接产生了跳跃点，而跳跃点包含了丰富的频率成分，因此dft算出来的频谱就不会是你以为的只有一根谱线，而是包含了周围的其他频率成分，如下图所示。这种现象称为频谱泄露。

当然，如果我们拿来计算DFT时截取的那段信号能够正巧使首尾拼接的话，那么算出来的就真的是一根谱线。

傅里叶与卷积

发表于 2018-11-24 | 更新于 2018-11-27 | 分类于 信号处理 |

两大主题

信号与系统的两大主题：

  • 信号处理
  • 系统分析

信号处理常用傅里叶分析，系统分析常用卷积。信号处理与系统分析看似不同领域，实际有很多共通之处，特别是通过卷积定理，可以将傅里叶分析与卷积联系起来，因此不管是分析信号还是系统都可以使用傅里叶分析与卷积;
当然信号处理与系统分析还有其他很多方法，比如信号处理中还有自相关，系统分析中还有拉普拉斯变换和Z变换 ,这些暂且按下不表，下面先介绍傅里叶与卷积。

傅里叶分析理论

傅里叶分析理论认为信号可以分解成许多不同频率的复正弦信号按照不同的权值组合而成。根据信号的不同，可以包括以下内容：

  • FS——连续时间周期信号的傅里叶
  • FT——连续时间非周期信号的傅里叶
  • DFS——离散时间周期信号的傅里叶
  • DTFT——离散时间非周期信号的傅里叶
  • DFT——离散信号的傅里叶
  • FFT——快速傅里叶

其中DFT是对DTFT的近似，FFT是计算DFT的快速算法。

傅里叶分析的要诀：
离散对周期，周期对离散；连续对非周期，非周期对连续；
因此根据以下的步骤可以很轻松写出傅里叶分析的公式

  • 基底为一组复正弦，找出这组复正弦的表达式$e^{j2 \pi 频率*时间}$，根据信号类型的不同，判断频率和时间是连续的，还是离散的
  • 如果是离散的，用$\sum$；如果是连续的，用$\int$
  • 如果是周期的，需要乘一个系数（周期的倒数）；如果是非周期的，则不用乘系数。时域和频域最多只有一个需要乘以系数；

FS

连续时间周期信号，周期为T，连续周期信号的频谱非周期离散，

  1. 频率是离散的，时间是连续的，则基频为$1 \over T$，其谐波频率为$k \over T$，基底为$e^{j2\pi {k \over T}t}$;
  2. 频谱是离散的，所以合成时用$\sum$，时间是连续的，所以分析时用$\int$;
  3. 时域是周期的，所以分析时时域积分长度为一个时间周期T，要乘以系数，周期的倒数；频域是非周期的，所以长度是$\infty$,不用乘以系数；

$$x(t) = \sum\limits_{ k = - \infty }^{ + \infty } {X({k \over T}){e^{j2\pi {k \over T}t}}} $$

$$X({k \over T}) = {1 \over T}\int_{ - {T \over 2}}^{ {T \over 2}} {x(t){e^{ - j2\pi {k \over T}t}}dt} $$

FT

连续时间非周期信号，连续非周期信号的频谱是非周期连续的

  1. 频率是连续的，时间是连续的，则基底为$e^{j2 \pi ft}$;
  2. 频谱是连续的，所以合成时用$\int$,时间是连续的，所以分析时用$\int$;
  3. 频谱和时间都是非周期的，所以积分长度都为$\infty$，都不用乘以系数。

$$x(t) = \int_{ - \infty }^\infty {X(f){e^{j2\pi ft}}df} $$
$$X(f) = \int_{ - \infty }^\infty {x(t){e^{ - j2\pi ft}}dt} $$

DFS

离散时间周期信号，采样间隔为Ts，一个周期有N个采样点，周期为T=NTs，离散时间周期信号的频谱是周期离散的

  1. 频率是离散的，时间是离散的，基频为周期的倒数$1 \over NT_s$，谐波频率为$k {1 \over NT_s}$,故基底为$e^{j2 \pi k {1 \over NT_s} nT_s}$;
  2. 频谱是离散的，所以合成时用$\sum$，时间是离散的，所以分析时用$\sum$;
  3. 频谱是周期的，所以合成时长度为一个频率周期，需要乘以系数；时域是周期的，所以分析时长度为一个时间周期，本来还需要乘以系数；不过因为时域和频域最多只需要一个乘以系数，前面合成已经乘了，所以分析就不用了

$$x(n{T_s}) = {1 \over N}\sum\limits_{k = 0}^{N - 1} {X(k){e^{j2\pi {k \over {N{T_s}}}n{T_s}}}} $$

$$X(k) = \sum\limits_{n = 0}^{N - 1} {x(n{T_s}){e^{ - j2\pi {k \over {NTs}}nTs}}} $$

DTFT

离散时间非周期信号， 离散时间非周期信号的频谱时周期连续的

  1. 频率是连续的，时间是离散的，基底为$e^{j2 \pi fnT_s}$;
  2. 频域是连续的，合成时用$\int$；时间是离散的，分析时用$\sum$;
  3. 频域是周期的，合成时长度为一个频率周期，需要乘以系数，为频率周期的倒数； 时域是非周期的，分析时不用乘以系数；
    频率周期是离散时域两个点的时间间隔的倒数。

$$x(n{T_s}) = {1 \over { {f_s}}}\int_{ - { { {f_s}} \over 2}}^{ { { {f_s}} \over 2}} {X(f){e^{j2\pi fn{T_s}}}df} $$

$$X(f) = \sum\limits_{n = - \infty }^{ + \infty } {x(n{T_s}){e^{ - j2\pi fnTs}}} $$

DFT

隐含假设离散时间周期信号，DFT实际上是对DFS时域和频域各取一个周期，用于近似DTFT，离散时间周期信号的频谱是周期离散的

$$x(n{T_s}) = {1 \over N}\sum\limits_{k = 0 }^{ N-1} {X(k){e^{j2\pi k{1 \over {NTs}}nTs}}} $$

$$X(k) = \sum\limits_{n = 0 }^{ N-1 } {x(n{T_s}){e^{ - j2\pi k{1 \over {NTs}}nTs}}} $$

DFT的原理
目标：我们想通过傅里叶变换了解信号，并且想借用计算机来帮助人们计算。
遇到的问题：计算机只能计算和存储有限长的序列。
解决方法：所以需要将信号的时域和频域进行采样离散化。而根据采样理论，时域的离散化会引起频域进行周期延拓。频域的离散化会引起时域的周期延拓。DFT正好使信号的时域延拓的周期等于信号的长度，频域延拓的周期等于采样频率。这一目标通过频域的一个周期的点数与时域的一个周期的点数相同实现的。

小结

这里写的公式跟书上不一样，原因在于，我坚持使用频率，而非其他什么角频率，圆周频率等作为频谱横坐标的表示，因此上面所有情况的频谱横坐标都是$f$，而非什么$\Omega$或者$\omega$。
全部用$f$才能更好地统一各种情况的频谱表示，才能理解频谱的真正含义。如果需要使用$\Omega$或者$\omega$的形式，可以查书，或者用关联表达式替换上面的$f$。

$$\Omega = 2 \pi f$$

$$\omega = 2 \pi f/f_s = 2 \pi fT_s$$

由于最后基本上都是用DFT或者FFT来分析，所以这里我整理一下DFT的公式

$$X(k) = \sum\limits_{n = 0 }^{ N-1 } {x(n){e^{ - j{2\pi \over N}kn}}} $$
$$x(n) = {1 \over N} \sum\limits_{n = 0 }^{ N-1 } {X(k){e^{ j{2\pi \over N}kn}}}$$

LTI系统和LSI系统

LTI系统连续的线性时不变系统，LSI系统是离散的线性移不变系统，有四种方法可以用来表达LTI系统或者LSI系统的特征

  • 脉冲响应
  • 频率响应
  • 传递函数(LTI系统)或者转移函数(LSI系统)
  • 状态空间表达式

不过这里主要讨论脉冲响应，频率响应将在卷积定理之后再介绍。

脉冲响应

对于LSI系统，知道单个脉冲引起的响应之后，根据LSI系统的线性性质和移不变性质，可以将输入信号x(n)看成是由一串脉冲序列，那么信号x(n)引起的输出就是这串脉冲序列引起的响应经过调幅,平移,叠加的信号。

如以下公式显示：
输入信号x(n)可以拆成一系列的脉冲，脉冲的大小为x(k)。
这里h(n)就是系统的脉冲响应，

$$\eqalign{
\delta (n) &\to h(n) \cr
\delta (n - k) &\to h(n - k) \cr
x(k)\delta (n - k) &\to x(k)h(n - k) \cr
\sum\limits_{k = - \infty }^{ + \infty } {x(k)\delta (n - k)} &\to \sum\limits_{k = - \infty }^{ + \infty } {x(k)h(n - k)} \cr} $$
从上面的公式引出了卷积的定义：

$$
x(n)*y(n)=\sum\limits_{k = - \infty }^{ + \infty } {x(k)y(n - k)}
$$

频率响应

$$H({e^{j\omega }}) = \sum\limits_{n = - \infty }^{ + \infty } {h(n){e^{ - j\omega n}}} $$
实际上就是说，LSI系统的频率响应是其脉冲响应的DTFT，因为在计算机中无法真正计算DTFT，通常用DFT近似DTFT，因此在计算机中可以用DFT来表示LSI系统的频率响应
$H(k) = DFT(h) = \sum\limits_{n = 0 }^{ N-1 } {h(n){e^{ - j{2\pi \over N}kn}}}$

卷积定理

卷积定理：

$$DFT\left( {f*g} \right){\rm{ }} = {\rm{ }}DFT\left( f \right) \cdot DFT(g)$$

用文字表述就是
卷积的傅里叶变换等价于傅里叶变换相乘;
时域的卷积等价于频域相乘;
频域的卷积等价于时域相乘;
卷积定理联系了卷积和傅里叶变换，极大地丰富了信号分析和系统分析的内容。

卷积定理的应用

理解时域加窗

给信号x(n)加窗w(n)，即y(n) = x(n)w(n);
根据卷积定理，时域相乘等价于频域卷积，因此Y(k) = X(k) * Wk);

理解频域加窗

给信号x(n)的频谱加方窗，频域相乘等价于时域卷积

理解LSI系统的频率响应

LSI系统的输入是x(n)，系统的脉冲响应是h(n)，则输出信号y(n)=x(n)*h(n)
可以将系统的脉冲响应h(n)看成是一个窗函数，其频谱为DFT(h)，根据卷积定理，时域卷积等价于频域相乘

$$DFT\left( {y} \right){\rm{ }} = {\rm{ }}DFT\left( x \right) \cdot DFT(h)$$
即$Y(k) = X(k)H(k)$

因此系统的频率响应就是

$$H(k) = DFT(h)$$

DFT背后隐含的信息

DFT隐含假设：信号是周期的，只不过在使用的时候，只拿其中一个周期出来计算。

所以我们计算信号的DFT时，要清楚算出来的频谱并不是这个信号的，而是这个信号扩展后的周期信号的。

基频与主频

fundamental frequency：周期信号最低的频率成分

dominant frequency：周期信号幅度最大的频率成分

通常人耳通过基频感受音调的，即使它不是幅度最大的频率成分

谐波结构

谐波：频率是基频的整数倍的成分T

频谱与谱图

signal：

wave: 将信号的时间轴ts和取值fs组合在一起，就是wave

spectrum:频谱

sepctrogram：谱图，随时间变化的频谱

时间分辨率和频率分辨率

Gabor limit: 提高时间分辨率就会降低频率分辨率，提高频率分辨率就是降低时间分辨率。

时间分辨率与频率分辨率互为倒数，因此无法同时提高时间分辨率和频率分辨率。

混叠现象

folding frequency：采样频率的一半。

混叠现象(aliasing)是数字信号处理中最重要的的现象之一，表示采样时高频信号被当成低频信号。例如，采样频率为10000Hz时，folding frequency是5000Hz，如果去采样5500Hz的正弦信号时，采样结果与采样4500Hz的正弦波一样，错误地以为信号是4500Hz，而非实际的5500Hz，将原本的高频信号当成了低频信号，即aliasing。

频谱泄漏现象

频谱泄漏：即在分析信号时，引入了其他本没有的频率成分。

就像我们在测量物体的长度时，并不能获得真实长度，而是得到测量长度，存在测量误差，但是可以通过更加精密的手段来减少测量误差。
我们在在分析信号的频谱时，也会不可避免的引入

比如有一个周期为440Hz的正弦波信号，使用计算机算它的频谱，由于计算机只能计算有限长度，因此，我们只能截取其中一段出来计算DFT，根据DFT的隐含假设，DFT实际计算的是所截那段的周期延拓信号的频谱。信号周期延拓时，如果前一周期的最后一个点与下一周期的第一个点不相等，那么就会出现跳跃点，而跳跃点通常包含丰富的频率成分，也就引入了实际信号没有的频率。

基本波形

三角波：只有奇次谐波，并且谐波的幅度与频率的平方成反比。
$A(f) = \frac{k}{f^2}$

方波：只有奇次谐波，并且谐波的幅度与频率成反比
$A(f) = \frac{k}{f}$

锯齿波：同时包含奇次谐波和偶次谐波，并且谐波的幅度与频率成反比
$A(f) = \frac{k}{f}$

抛物线波：同时包含奇次谐波和偶次谐波，并且谐波的幅度与频率的平方成反比
$A(f) = \frac{k}{f^2}$

信号与波形

信号与波形的区别：
信号：只有时间序列及对应的值，但不知道相邻两个信号点的实际时间间隔。

波形：带有时间戳的信号。给定时间序列和采样频率，就可以得到实际的时间戳。

以视频字幕来比喻，信号就是单纯的字幕文字，而波形就是带了时间戳的字幕，将字幕与实际的讲话时间对应上。

谐波结构

周期信号是由基频和谐波组成的。

基频(fundamental frequency)：基频是周期信号最低的频率成分。

主频(dominant frequency)：主频是周期信号幅值最大的频率成分。

这里第一个频率既是基频，也是主频。而有些信号的基频和主频并不是同一个频率。

四种基本波形的谐波结构

方波，三角波，锯齿波，抛物线波，及其频谱特性

  • 方波：只包含奇次谐波，并且谐波的幅度与频率成反比，$A(f) = \frac{k}{f}$
  • 三角波：只有奇次谐波，并且谐波的幅度与频率的平方成反比，$A(f) = \frac{k}{f^2}$
  • 锯齿波：同时包含奇次谐波和偶次谐波，并且谐波的幅度与频率成反比，$A(f) = \frac{k}{f}$
  • 抛物线波：同时包含奇次谐波和偶次谐波，并且谐波的幅度与频率的平方成反比，$A(f) = \frac{k}{f^2}$

什么时候只包含奇次谐波，什么时候只包含偶次谐波？

  • 如果将信号平移半个周期与原来的波形正好相反，那么只包含奇次谐波，
  • 如果将信号平移半个周期与原来的波形正好重合，那么只包含偶次谐波

周期卷积与线性卷积

如果将周期为N的周期序列$\tilde x(n)$ 和$\tilde y(n)$的主值序列x(n)和y(n) 作线性卷积，然后再讲线性卷积序列以N为周期进行周期延拓，得到的周期序列就是两个周期序列的周期卷积。

$$
\begin {split}
x(n) &= \tilde x(n) \cdot R_N(n), y(n) &= \tilde y(n) \cdot R_N(n) \
g(n) &= x(n) y(n), \tilde f(n) &= \tilde x(n) \tilde y(n) \
\tilde f(n) &= \sum\limits_{i = =\infty} ^{+\infty} {g(n -iN)} \
\end {split}
$$
