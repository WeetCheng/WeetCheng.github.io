傅变拉变Z变

发表于 2018-11-05 | 更新于 2018-11-27 | 分类于 信号处理 |

前言

拉普拉斯变换和傅里叶变换都是针对连续信号的；

拉普拉斯变换是傅里叶变换的推广；

傅里叶变换是从频域的角度看待信号，拉普拉斯变换是从复频域的角度看待信号；

傅里叶变换有一个限制条件就是：信号必须绝对可积；

拉普拉斯变换为了弥补这一缺陷，先将信号乘以衰减指数$${e^{ - \beta t }}$$，从而保证信号绝对可积；

Z变换相当于离散信号的拉普拉斯变换；

Z变换单位圆上的点相当于离散信号的傅里叶变换;

Z变换中的Z平面与拉普拉斯的S平面存在映射关系，$z = e^{Ts}$;

正态分布和均匀分布

发表于 2018-11-01 | 更新于 2018-11-27 | 分类于 信号处理 |

# 正态分布

randn()：产生正态分布的随机数或随机矩阵的函数，生成的标准正态分布，即期望为0，方差为1

randn(N): 产生N * N的随机矩阵

randn(M, N): 产生 M * N的随机矩阵

randn(M, N, P, …): 产生M N P * …的随机矩阵

randn(size(A)): 产生与矩阵A相同大小的随机矩阵

example

y = randn(1000, 1) % 产生一维，长度为1000的随机数

如果想生成非标准正态分布，可以借用正态分布的一个性质来实现。

$$
X{\sim}N(\mu ,{\sigma ^2}) \
Y = aX + b{\sim} N(a\mu + b, {a^2 \sigma ^2})
$$

正态分布标准化公式

$$
X{\sim}N(\mu, \sigma ^2) \
{ {X-\mu}\over{\sigma}} \sim N(0, 1)
$$

# 均匀分布

rand(): 产生均匀分布的随机数或随机矩阵的函数

rand(N): 产生N * N的随机矩阵

rand(M, N): 产生 M * N的随机矩阵

rand(M, N, P, …): 产生M N P * …的随机矩阵

rand(size(A)): 产生与矩阵A相同大小的随机矩阵

数组和指针的区别

发表于 2018-10-18 | 分类于 编程 |

数组和指针在某些地方看上去很相似，但是实际上两者本质上是不同的。

  • 数组是一块连续的内存单元的名称，这块内存单元存放着数据。
  • 指针则是一个内存单元的名称，这个内存单元存放的是另一个内存单元的地址。

根据这个不同可以解释很多现象，比如指针占用的内存大小是固定的，即为机器的位数；而数组占用的内存空间可以自定义的。

假设定义了一个二维数组a，如下：

那么

  • a——代表二维数组本身，即连续 6个int大小的内存单元。本质是一个数组
  • a+1——a原本是二维数组，运算表达式中先被转换成指针，指向a的第一个元素，然后指针加1，则a+1指向二维数组a的第2个元素。本质是一个指针
  • a[1]——代表二维数组a的第2个元素，即一维数组。本质是一个数组
  • a[0]+1——a[0]本来是一个一维数组，运算表达式中先被转换成一个指针，指向a[0]的第1个元素，然后指针加1，则a[0]+1指向一维数组a[0]的第2个元素。本质是一个指针
  • a[0][0]——代表第一个内存单元。本质是一个变量
